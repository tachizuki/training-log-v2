## 日本語

### タイトル（30字以内）
- ★確定: `PhysiqueLog 体重・筋トレ記録 大会減量`（25字／差別化語「大会」で上位を狙う）
- A/Bテスト候補（広め）: `PhysiqueLog 体重・筋トレ記録 ダイエット`
- ※「（フィジークログ）」は削除（検索されない死にスペース）
- 取れる主な語: 体重記録 / 筋トレ記録 / 大会 / 減量 / 体重 / 筋トレ / 記録

### 簡単な説明（80字以内）
体重・筋トレ・食事(PFC)を記録するダイエット＆体重管理アプリ。減量や大会準備の体重変化を要因分析し、週ごとの目標体重まで導きます。

### 詳しい説明
PhysiqueLog（フィジークログ）は、体重記録・筋トレ記録・食事(PFC)管理を1つにまとめた、ダイエット／ボディメイク向けの記録アプリです。減量・増量から大会（フィジーク・ベストボディ）準備まで、「なぜ体重が増えた・減ったのか」が分かる体重変動分析で、目標達成までの毎日をサポートします。

▼ 無料でできること
・体重記録と体重グラフ（体重推移がひと目で分かる）
・筋トレ記録（部位別カテゴリ・種目検索・カスタム種目／レストタイマー／履歴カレンダー）
・食事のカロリー・PFC（タンパク質・脂質・炭水化物）・塩分の記録
・水分・睡眠・歩数の記録
・体重変動分析：前日の塩分・水分・炭水化物・カロリー・筋トレ・睡眠から増減の要因を提示
・全期間の履歴閲覧／データのエクスポート・インポート（JSONバックアップ）

▼ プレミアム（大会に出る人・本格的に追い込む人へ。初回14日間無料）
・🗺 大会ロードマップ：大会日から逆算し、目標体重を週ごと・月ごとに按分。目標 vs 実績が一目で分かる
・📈 体重グラフに日毎の目標体重ライン
・📋 週次レポート
・💪 筋トレ分析（推定1RMの推移・停滞検知）
・☁️ 複数端末のクラウド同期

▼ こんな方に
・体重を記録して、ダイエット・減量を計画的に進めたい
・筋トレ（トレーニング）の重量・レップを記録し、停滞を早く見つけたい
・大会（フィジーク・コンテスト）に向けて逆算で体重管理したい
・体重の上下に振り回されず、増減の要因を理解してコントロールしたい

体重記録・筋トレ記録・全期間の履歴はずっと無料。まずは今日の体重から、体重管理を始めましょう。

（プレミアムのご購入・解約は、アプリのログインアカウントではなく、ご利用端末のGoogle Playアカウントに登録されます。複数のGoogleアカウントをお使いの場合はPlayストアのアカウントをご確認ください。）

## 運用ルール
- 詰め込み（キーワード羅列）はポリシー違反＆逆効果。自然な反復2〜3回まで。
- 日本語と英語は別索引 → 英語版も別途最適化。
- タイトル/アイコン/簡単な説明は Play Console「ストアの掲載情報のテスト」(A/B) で比較。まずタイトルをテスト推奨。
- 反映後は Console →「ユーザー獲得」→ 検索キーワード を2〜4週ごとに確認し改善。
- 順位は文言だけでなく 評価・DL数・継続率 にも左右される。
