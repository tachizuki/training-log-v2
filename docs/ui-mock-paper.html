「稽古帳」— ゼロベース設計の趣意

  • メタファーを「ガジェット」から「紙のノート」へ。競技者は元々トレーニングノートをつける文化を持つ。アプリを"光る計器"ではなく"毎日書き込む帳面"として設計し直した。
  • 紙×墨×朱の三色。カードUI・影・グラデーションを全廃し、罫線と余白だけで構造を作る。アクセントは朱印の一色のみ。
  • 1日=1ページ。ホーム画面という概念をなくし、開いたら今日のページ。前日・翌日へは帳面をめくるように移動。
  • 水分は「正」の字タリー。ボタン連打のmlカウンタより、一杯ごとに画数が増えていく方が記録の手応えがある。
  • 記録を締める「済印」。1日の記録を終えたら朱印を押す。継続のごほうびをバッジやストリーク数字ではなく、所作として与える。
  • 数字はすべて等幅フォント。台帳らしく桁が揃い、kg×repの一覧性が上がる。見出しは明朝で引き締める。
  • ナビは文字のみ4項目。アイコンを排し「今日・稽古帳・推移・大会」。競技者アプリの核である大会を一等地に。
  • 休憩タイマーも帳面の一行。モーダルやFABではなく、ページ下端に挟まった栞のように常駐。
  • 留意点（正直な評価）: 明所での視認性は上がるが、暗いジムではダークテーマに分がある。実装するなら紙テーマ／墨テーマ（ダーク反転）の切替が現実解。
