UX主導の再設計 — 何を変えたか

  • 保存ボタンの廃止。全項目が自動保存（ヘッダーに常時「●自動保存」表示）。「保存バーがキーボードに隠れる」「保存バーが残る」系の問題はUIの工夫ではなく概念ごと削除して解決。
  • OSキーボードを使わない。数値入力は下半分の専用キーパッド。±2.5kg/±0.5kgのステップと「前回値」が1タップ。キーボードがUIを隠す問題自体が起きない。ジムで手袋・汗ありでも押せる大型キー。
  • ホーム＝今日のチェックリスト。体重・睡眠・食事・水分・筋トレが「済/未」の行で並ぶ。アプリを開く理由は毎日「今日の記録を埋める」ことなので、それをそのままUIにした。未記録の行をタップ→その場で入力→行が✓になる。
  • 水分は画面遷移なし。+250/+500/+1Lをホーム上で完結。専用画面もタブも持たない。
  • セット✓でレストタイマー自動開始。「タイマーを開いて押す」操作を削除。バーは常駐し+30s/スキップのみ。
  • 「前回コピー」を種目ごとに。筋トレ記録の実態は前回の微修正。3セット分が1タップで入り、変わった所だけ直す。
  • データ画面は結論ファースト。グラフの前に「順調か否か・着地予測」を日本語で表示。数字の羅列を読ませない。
  • 大会コンテキストを全画面のヘッダーに常設。D-47と残りkgがどの画面でも見える。競技者の意思決定は常に大会基準だから。
  • 雰囲気: ジム（暗所）で使う前提でダークは維持。ただし彩度を落とし、アクセント1色＋状態色（緑/黄/赤）のみ。数字は等幅・大サイズで瞬読性優先。
  • ナビは4つ。記録タブを廃止（ホームが記録そのものなので）。今日・筋トレ・データ・設定。
