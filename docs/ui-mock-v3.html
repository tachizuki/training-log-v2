PhysiqueLog v3 — 再設計の要点

  • ホーム = ダッシュボード化。ロゴ中心のスプラッシュをやめ、今日のターゲット進捗（kcal/P/水分/睡眠）・体重トレンド・今日のワークアウトを一画面に集約。アプリを開いた瞬間に「今日やること」が分かる。
  • 大会カウントダウンをヒーローに昇格。競技者向けアプリの核なので、リング型プログレス＋減量フェーズ＋目標までの残量を最上部に常設。
  • ナビを6→4タブ＋中央FABに削減。「水分」タブは廃止し、ホームのクイックチップとFABシートに統合（水分記録は1タップ動作であり、専用画面は過剰）。「設定」はナビから外しホーム右上へ。
  • クイック記録FAB。最頻出の4操作（体重・食事・水分・筋トレ開始）をどの画面からも2タップで。
  • 記録画面をセグメント式に。長い縦フォームをチップで章分けし、スクロール量を削減。保存バーはナビ直上に固定。
  • 筋トレ画面はセッション主体。セット表に1RM列と完了チェックを統合、前回記録をチップでインライン表示。レストタイマーは画面遷移なしの常駐ミニバー（リング進捗＋スキップ/+30s）。
  • 絵文字 → ラインSVGアイコン。ナビと主要アクションの視覚ノイズを低減し、プロ向けの精悍さを出す。
  • カラー微調整。#e8ff00 → #d8ff3e（眩しさ軽減）、サブカラー階層（text-sub/muted）を追加して情報密度を上げても読めるように。
  • ストリーク🔥で継続動機付け。連続記録日数をヘッダーに常設。
