# Google Play Store 掲載情報

---

## アプリ名
TrainingLog - 大会競技者向け記録アプリ

## 簡単な説明（80文字以内）
体重・食事・水分・筋トレを一括管理。大会に向けた減量を科学的にサポート。

## 詳細な説明（4000文字以内）

フィジーク・ボディビル・クラシックフィジークなど、大会を目指すすべての競技者のために設計されたトレーニングログアプリです。

━━━━━━━━━━━━━━━
📊 毎日の記録
━━━━━━━━━━━━━━━
・体重（0.1kg単位）
・食事：カロリー・タンパク質・脂質・炭水化物・塩分
・水分摂取量（リアルタイムプログレスリング）
・睡眠時間・歩数・日サロ時間
・筋トレ（種目・重量・セット数・レップ数）
・有酸素運動（METベースのカロリー計算）

━━━━━━━━━━━━━━━
🔍 体重変動分析
━━━━━━━━━━━━━━━
前日の塩分・水分・カロリー収支・炭水化物・筋トレ・睡眠などを自動分析し、「なぜ今日の体重がこうなったのか」を13項目で解説します。減量中の体重の上下に一喜一憂しなくなります。

━━━━━━━━━━━━━━━
🗺 大会ロードマップ（プレミアム）
━━━━━━━━━━━━━━━
大会日と目標体重を設定すると、週ごとの目標体重ラインを自動計算。実績と重ねて表示することで、ペースが遅れているかどうかを一目で確認できます。週・月・全体の3つの視点で確認可能。

━━━━━━━━━━━━━━━
📋 週レポート（プレミアム）
━━━━━━━━━━━━━━━
週平均体重・ジム回数・平均カロリー・平均タンパク質・平均水分摂取を1画面で確認。週目標との差分をリアルタイム表示。

━━━━━━━━━━━━━━━
☁️ クラウド同期
━━━━━━━━━━━━━━━
Googleアカウントでログインすると、データがクラウドに自動保存されます。機種変更時もデータが引き継がれます（1端末無料）。

━━━━━━━━━━━━━━━
🔔 毎日通知
━━━━━━━━━━━━━━━
設定した時間に「今日の記録はしましたか？」と通知。記録の習慣化をサポートします。

━━━━━━━━━━━━━━━
💪 筋トレ機能
━━━━━━━━━━━━━━━
・RESTタイマー・ストップウォッチ内蔵
・前回のトレーニング記録を参照しながら入力
・マイセット機能でよく使う種目構成を保存
・トレーニング結果を画像でシェア

━━━━━━━━━━━━━━━
プレミアムプランについて
━━━━━━━━━━━━━━━
無料プランでは直近30日の記録を閲覧できます。
プレミアムプラン（月額300円）では全期間の履歴閲覧・大会ロードマップ・週レポート・複数端末クラウド同期が利用できます。

## データの取り扱い

- **暗号化して転送しているか？** → はい（HTTPS/TLS）
- **ユーザーがデータの削除を要求できるか？** → はい
- **データはデバイス上のみに保存されるか？** → 選択可能（ログインしない場合はローカルのみ）

# スクリーンショット撮影チェックリスト

Play Storeには最低2枚（推奨8枚）のスクリーンショットが必要です。
以下の画面を撮影してください（縦向き、1080×1920px以上推奨）。

1. **ホーム画面** - 大会カウントダウンが表示された状態
2. **記録画面** - 体重・食事・塩分など入力フォーム
3. **水分記録画面** - プログレスリング
4. **筋トレ画面** - 種目・セット・重量の記録
5. **グラフ画面** - 体重推移グラフ
6. **体重変動分析** - 分析カードを展開した状態
7. **大会ロードマップ** - グラフ表示
8. **週レポート** - 数値が入っている状態

# リリースビルド手順

Android Studioで以下の手順でApp Bundleを生成してください。

1. Build → Generate Signed Bundle / APK
2. Android App Bundle を選択
3. キーストアを新規作成（または既存のものを使用）
   - キーストアファイルは絶対に紛失しないこと（バックアップ必須）
4. リリースビルドを生成
5. app/release/app-release.aab をPlay Consoleにアップロード
